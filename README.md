# Neural Networks - NTUA ECE

This repository contains my work for the **Neural Networks** course during my studies at the **National Technical University of Athens (NTUA), School of Electrical and Computer Engineering (ECE)**.

## Contents

### Labs
- **neural-networks-lab1** — First laboratory assignment covering introductory neural network concepts and experiments using the UCI dataset and Kaggle datasets.
- **neural-networks-lab2** — Second laboratory assignment focused on Unsupervised Learning.
- **neural-networks-lab3** — Third laboratory assignment.

### Course Material
Lecture slides, lab notebooks, and past exam papers used for studying, covering topics such as:
- Perceptron & MLP
- SVM
- Clustering (k-Means, Hierarchical, SOM)
- Genetic Algorithms
- Deep Learning & CNNs
- RNNs
- Text Mining & Word Embeddings

### neural_networks_ntua
Additional notebooks and assignments related to the course labs.
